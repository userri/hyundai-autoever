# 현대 오토에버 회사 소개 웹사이트
현대 오토에버의 서비스와 기술력을 소개하는 공식 웹사이트입니다.
## 페이지 구성
- **index.html**: 메인 페이지
- **about.html**: 회사 소개
- **services.html**: 서비스 소개
- **contact.html**: 연락처
## 기술 스택
- HTML5
- CSS3
- JavaScript
## 실행 방법
1. 프로젝트 클론
```bash
 git clone https://github.com/username/hyundai-autoever.git
```
2. 폴더 이동
```bash
 cd hyundai-autoever
```
3. 브라우저에서 index.html 열기
 - VSCode Live Server 사용 권장

  - 또는 더블 클릭으로 브라우저에서 열기
